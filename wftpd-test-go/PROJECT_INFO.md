# WFTPD FTP/SFTP 测试工具 - Go 语言实现

此目录包含使用 Go 语言开发的 WFTPD 服务器测试工具，提供了比 Python 脚本更稳定的测试环境。

## 文件清单

### 源代码文件
- `main.go` - 主程序入口，包含 FTP 测试模块
- `sftp_test.go` - SFTP 测试功能实现

### 配置文件
- `go.mod` - Go 模块定义和依赖管理
- `go.sum` - 依赖校验和文件

### 构建和部署文件
- `build.bat` - Windows 批处理构建脚本
- `USAGE.md` - 详细的使用说明文档
- `README.md` - 项目概述文档

### 测试数据目录
- `testdata/` - 自动生成的测试文件存放目录

## 依赖关系

此项目依赖以下 Go 包：
- `golang.org/x/crypto/ssh` - SSH 协议实现（用于 SFTP）
- `github.com/pkg/sftp` - SFTP 客户端协议实现

## 功能模块

### FTP 测试模块 (`main.go`)
- 基本连接测试
- 用户认证测试  
- 目录操作测试
- 文件上传/下载测试
- 断点续传测试
- 被动模式测试
- TLS 加密测试

### SFTP 测试模块 (`sftp_test.go`)
- SSH 连接测试
- SFTP 协议握手
- 目录操作测试
- 文件操作测试
- 文件重命名测试
- 文件信息查询

## 编译说明

要编译此项目，需要：

1. 安装 Go 1.21 或更高版本
2. 在项目目录中运行 `go mod tidy` 下载依赖
3. 运行 `go build` 生成可执行文件

## 测试流程

程序会自动执行以下测试流程：
1. 生成测试文件
2. 运行 FTP 测试套件
3. 运行 SFTP 测试套件
4. 生成测试报告

## 日志和输出

所有测试操作都会输出详细日志，包括：
- 操作描述
- 服务器响应
- 操作耗时
- 成功/失败状态

## 特点

- 符合 FTP/SFTP 相关 RFC 标准
- 详细的日志记录
- 每个操作返回响应码
- Windows 控制台程序
- 高性能和稳定性
- 避免了 Python 脚本可能出现的兼容性问题

## 部署说明

将此目录放在任意位置，确保有足够权限创建测试文件即可运行。
