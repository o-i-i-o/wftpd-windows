# WFTPG FTP/SFTP 测试套件

## 📦 文件清单

已创建以下测试相关文件：

### 核心文件
1. **test_ftp_sftp.py** - 主测试脚本
   - 完整的 FTP 功能测试（18 项）
   - 完整的 SFTP 功能测试（16 项）
   - 自动启动/停止 wftpd 服务
   - 详细的测试结果统计

2. **requirements-test.txt** - Python 依赖文件
   - paramiko (SFTP 客户端库)

### 启动脚本
3. **run_test.bat** - Windows 批处理启动器
   - 自动检查 Python 和依赖
   - 一键运行测试

4. **run_test.ps1** - PowerShell 启动器（推荐）
   - 彩色输出
   - 智能检查结果摘要
   - 交互式查看测试报告

### 文档文件
5. **README_TEST.md** - 快速入门指南
   - 简明的使用说明
   - 快速开始步骤

6. **TEST_GUIDE.md** - 详细使用文档
   - 完整的测试说明
   - 每个测试项目的详细描述
   - 故障排除指南
   - 自定义测试方法

### 配置文件
7. **test_config.example.json** - 配置示例
   - 展示可配置的参数
   - 用于自定义测试设置

## 🚀 快速开始

### 方法 1：使用 PowerShell 启动器（推荐）
```powershell
.\run_test.ps1
```

### 方法 2：使用批处理启动器
```cmd
run_test.bat
```

### 方法 3：直接运行 Python 脚本
```bash
python test_ftp_sftp.py
```

## 🔧 自定义配置

### 修改测试参数

编辑 `test_ftp_sftp.py`：

```python
# 在 main() 函数中修改
ftp_tester = FTPTester(
    host='192.168.1.100',  # 修改 IP
    port=2121,             # 修改端口
    username='testuser',   # 修改用户名
    password='testpass'    # 修改密码
)

sftp_tester = SFTPTester(
    host='192.168.1.100',
    port=22222,
    username='testuser',
    password='testpass'
)
```

```python

result = ftp.run_all_tests()
print(f"FTP 通过率：{result.passed}/{result.total}")
```

### 2. 单独运行 SFTP 测试


## ⚠️ 注意事项

1. **管理员权限**: 某些测试可能需要管理员权限
2. **端口占用**: 确保端口 21 和 2222 未被占用
3. **防火墙**: 临时关闭防火墙或添加例外
4. **测试清理**: 测试完成后会自动清理
5. **并发测试**: 不要同时运行多个测试实例

## 🐛 常见问题

### Q: 连接失败怎么办？
A: 
1. 确认 wftpd 服务正在运行
2. 检查防火墙设置
3. 验证端口配置

### Q: 认证失败？
A:
1. 确认用户存在且密码正确
2. 检查 users.json 配置
3. 确保用户状态为 enabled

### Q: 如何查看详细日志？
A:
查看日志目录：`C:\ProgramData\wftpg\logs\`
